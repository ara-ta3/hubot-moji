# hubot-moji

## Install

```
npm install npm-moji
```

## Usage

```
env HUBOT_MOJI_ZEROONEMAP_CONF=`pwd`/conf/conf.json ./bin/hubot
```

## Conf Example

```
{
    "0": ":black_small_square:",
    "1":":star:"
}
```
